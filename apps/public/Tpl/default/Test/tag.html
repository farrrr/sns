		呼叫：W('Tag', array('width'=>'500px', 'appname'=>'support','apptable'=>'support','row_id'=>0,'tpl'=>'show','tag_url'=>'aaa','name'=>'public'))
		參數說明：
		width 		wigdet顯示寬度
		appname		資源對應的應用名稱
		apptable 	資源對應的表
		row_id 		資源在資源所在表中的主鍵ID,為0表示該資源為新加資源，需要在資源添加,往sociax_app_tag表添加相關資料
		tpl			顯示模版，tag/show 默認是tag，如果為show表示只顯示標籤
		tag_url		標簽上的連結字首，為空表示標籤沒有連結，只針對tpl=show有效
		name 		輸入框的input名稱,標籤的ID存儲的隱藏域名稱為 {name}_tags
