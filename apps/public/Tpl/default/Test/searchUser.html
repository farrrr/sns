搜索使用者：{:W('SearchUser', array('name' => 'selectUser', 'uids' => array(10000,14983),'follow'=>0,'max'=>10,'editable'=>1))} 
呼叫：W('SearchUser', array('name' => 'selectUser', 'uids' => array(10000,14983),'follow'=>0,'max'=>10,'editable'=>0))
參數說明：
name 	存儲uid的input名稱
uids	已選擇的使用者id集合
follow  是否只能選擇已關注的人（0表示可以選擇全部使用者，1表示只能選擇已關注的人）
max		最多可選擇的使用者個數
editable是否可修改選擇的結果，如果為0則不能選擇使用者，不能刪除已經選擇的使用者
